# Домашнє завдання 2

## Встановлення оточення

```bash
python3 -m venv venv
pip install -r requirements.txt
```

## Опис

Завдання складається з двох частин:

1. `image_classification.ipynb` - завдання з класифікації i ознак
2. `panorama_stitching.ipynb` - зшивання панорам із зображень.

Опис завдань наявний в самих ноутбуках. Якщо ви не знайомі з Jupyter Notebook, рекомендую ознайомитись із [матеріалом](https://www.youtube.com/watch?v=5pf0_bpNbkw)
